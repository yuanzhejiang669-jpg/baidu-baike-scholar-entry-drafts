# 方国光

义项名：新加坡科技设计大学校长、岩土工程学者

方国光（Phoon Kok Kwang），男，岩土工程学者，新加坡工程院院士、新加坡国家科学院院士、新加坡科技设计大学校长、Cheng Tsang Man讲席教授、AI Singapore副执行主席（研究）。\[3]\[6]\[13]

方国光在新加坡国立大学取得工程学学士、工程学硕士，在美国康奈尔大学取得博士；1995年回到新加坡国立大学工程学院任教；2013年—2015年任新加坡国立大学土木与环境工程系主任；2021年2月加入新加坡科技设计大学任教务长；2024年8月1日任新加坡科技设计大学校长；2025年8月1日受聘为AI Singapore副执行主席（研究）。\[3]\[4]\[5]\[6]\[9]

方国光的研究方向为岩土工程可靠度、风险分析、岩土参数统计表征、数据驱动岩土工程和机器学习方法；其成果推动了岩土工程不确定性量化、可靠度设计和风险决策研究进展。\[3]\[7]\[9]

# 人物经历

方国光在新加坡国立大学取得工程学学士、工程学硕士，在美国康奈尔大学取得博士。\[3]\[4]

1995年，回到新加坡国立大学工程学院任教。\[9]

2013年—2015年，任新加坡国立大学土木与环境工程系主任。\[4]

2015年，主持新加坡国立大学继续与专业教育工作组，推动成立新加坡国立大学持续与终身教育学院。\[4]

2020年11月，新加坡科技设计大学宣布方国光将任该校教务长；其时方国光任新加坡国立大学高级副教务长（学术事务）和杰出教授。\[4]

2021年2月，加入新加坡科技设计大学任教务长。\[4]

2024年8月1日，任新加坡科技设计大学校长。\[5]

2025年8月1日，受聘为AI Singapore副执行主席（研究），任期两年。\[6]

# 主要成就

## 科研成就

- **科研综述**

  方国光长期从事岩土工程可靠度、岩土工程风险决策和数据中心岩土工程研究，重点发展统计方法、数据驱动方法和机器学习方法，用于岩土工程参数表征、场地识别、模型不确定性量化和可靠度设计。其关于基于可靠度设计的研究影响了ISO2394:2015 General principles on reliability for structures等规范；其研究也将岩土工程风险控制从参数统计和可靠度计算扩展到数据库、机器学习和数字化决策。\[3]\[7]\[9]\[11]\[12]

# 人物评价
