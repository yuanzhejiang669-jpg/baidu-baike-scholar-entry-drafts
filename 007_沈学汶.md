# 沈学汶

义项名：美国国家工程院院士、加州大学伯克利分校环境工程荣休教授

沈学汶（Hsieh Wen Shen，1931年7月13日—2021年12月2日），男，生于中国北京，美籍华裔水利工程学家，美国国家工程院院士，加州大学伯克利分校环境工程荣休教授。\[1]\[2]\[3]

沈学汶中学阶段在中国上海就读，后入密歇根大学学习工程；1956年与克莱尔（Clare）结婚；1957年迁居伯克利；1961年获得加州大学伯克利分校土木工程博士学位；1985年加入加州大学伯克利分校，此前曾在科罗拉多州立大学任教18年；1993年当选为美国国家工程院院士；2021年12月2日逝世。\[3]\[6]

沈学汶的研究方向为泥沙工程、河流动力学、水资源、环境水力学、河流力学和基础泥沙输移；其成果推动了水库生态调度、河流修复和河流动力学研究的发展。\[2]\[3]\[6]\[8]

# 人物生平

1931年7月13日，沈学汶生于中国北京，中学阶段在中国上海就读；其后入密歇根大学学习工程。\[3]

1956年，与克莱尔结婚。\[3]

1957年，迁居伯克利。\[3]

1961年，获得加州大学伯克利分校土木工程博士学位。\[3]\[6]

1985年，加入加州大学伯克利分校；此前曾在科罗拉多州立大学任教18年，并曾任该校水文学与水资源项目负责人。\[3]\[6]

1993年，当选为美国国家工程院院士。\[3]

2021年12月2日，逝世，享年90岁。\[3]

# 主要成就

## 科研成就

- **科研综述**

  沈学汶长期从事泥沙工程、河流动力学、水资源、环境水力学、河流力学和基础泥沙输移研究。他围绕河流泥沙运动、冲积河道弯曲、水库放水系统、河流生态修复等问题开展研究，将河流工程与生态约束相结合，在水库流量控制和放水计划、基西米河修复研究、旧金山湾环境研究等方面形成成果。\[2]\[3]\[4]\[6]\[7]

- **科研项目**

  | 项目/研究 | 内容 |
  | :--- | :--- |
  | 基西米河修复研究 | 围绕美国佛罗里达州南部基西米河修复，评估备选修复方案是否达到生态目标，相关分析结合现场资料、物理模型和数值模型，并建议回填C-38运河部分河段。\[7]\[9] |
  | 旧金山湾模型研究 | 推动加州大学与美国陆军工程兵团达成协议，使大学工程师可使用索萨利托一英亩海湾模型开展研究，并为旧金山湾环境研究奠定基础。\[4] |

## 技术成就

沈学汶参与美国佛罗里达州南部基西米河修复工作，与工程师和生态学家团队服务于恢复鸟类、鱼类和河岸植物栖息环境；相关修复工作持续推进并取得实质进展。\[4]\[7]

在水库生态调度方面，沈学汶的研究侧重于开发满足下游生物群落生态需求的水库放水系统。美国国家工程院因其在制定水库流量控制和放水计划、恢复并增强河流生态环境方面的工作而授予其院士身份。\[3]\[6]

# 个人生活

- **家庭生活**

  1956年，沈学汶与克莱尔结婚；两人育有儿子安东尼（Anthony）、女儿伊夫琳（Eveline）及两个孙辈。沈学汶的著作《江水悠悠——水利工程学家治水记》从其外祖父詹天佑及早期留美学生写起，亦书写了几代华人赴美求学的经历。\[1]\[3]

# 人物评价

沈学汶是水利工程界的资深学者及顾问，对全球水资源问题有宏观视野和见解，并有丰富实践经验，其著作知识丰富且具权威性。*（中国香港商务印书馆评）*\[1]

沈学汶是一名有成就的教师和研究者，是加州大学伯克利分校土木与环境工程系的重要人物；其研究对土木与环境工程领域产生了重要影响*（“An accomplished teacher and researcher”；“tremendous impact”）*。*（加州大学伯克利分校土木与环境工程系评）*\[5]
