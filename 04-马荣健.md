# 马荣健

义项名：中国香港中文大学电子工程学系教授

马荣健（Wing-Kin Ma），男，国际电气与电子工程师协会会士（IEEE Fellow），中国香港中文大学电子工程学系教授。\[1\]\[3\]\[4\]

马荣健于1995年毕业于英国朴次茅斯大学电气与电子工程专业，获一等荣誉工程学士学位；1997年获中国香港中文大学电子工程哲学硕士学位；2001年获中国香港中文大学电子工程博士学位；成为教员前，曾在加拿大麦克马斯特大学、中国香港中文大学和澳大利亚墨尔本大学担任研究职位；2005年—2007年，任中国台湾清华大学通讯工程研究所助理教授，并兼任该校兼职职位；2017年当选为国际电气与电子工程师协会会士。\[1\]\[3\]\[4\]

马荣健的研究方向为信号处理、通信、优化、MIMO通信、盲源分离及高光谱遥感信号处理；其成果推动了信号处理与通信优化、高光谱解混、MIMO收发优化等领域的研究进展；曾获IEEE Signal Processing Magazine最佳论文奖、IEEE Signal Processing Letters最佳论文奖、IEEE信号处理学会最佳论文奖等荣誉，并当选为国际电气与电子工程师协会会士。\[1\]\[3\]\[4\]\[5\]\[6\]

# 人物经历

1995年，马荣健毕业于英国朴次茅斯大学电气与电子工程专业，获一等荣誉工程学士学位。\[3\]\[4\]

1997年，获中国香港中文大学电子工程哲学硕士学位。\[3\]\[4\]

2001年，获中国香港中文大学电子工程博士学位；成为教员前，曾在加拿大麦克马斯特大学、中国香港中文大学和澳大利亚墨尔本大学担任研究职位。\[3\]\[4\]

2005年—2007年，任中国台湾清华大学通讯工程研究所助理教授，并兼任该校兼职职位。\[3\]\[4\]

2017年，当选为国际电气与电子工程师协会会士。\[1\]\[3\]

# 主要成就

## 科研成就

- **科研综述**

  马荣健主要从事信号处理、通信和优化应用研究，研究内容涉及优化、MIMO通信、盲源分离及其在语音和生物医学图像中的应用，以及高光谱遥感信号处理。他围绕无线通信中的MIMO检测、复杂移动网络中的干扰管理和资源优化、面向生物医学图像和高光谱遥感图像的矩阵分解等问题开展研究，其工作将优化方法与信号处理、通信系统和遥感数据分析相结合。\[1\]\[3\]\[4\]

  在高光谱解混方向，马荣健从信号处理视角研究高光谱遥感中的材料识别与组成估计问题，将其与盲源分离、优化和机器学习方法联系起来；其相关报告和论文围绕高光谱传感器的高光谱维度、解混理论与算法，以及高光谱图像在遥感、生物医学图像和计算机视觉等场景中的方法联系展开。\[2\]\[7\]
