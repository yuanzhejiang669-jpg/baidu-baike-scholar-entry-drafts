# 王佑曾

义项名：美国国家工程院院士、美国艺术与科学院院士

王佑曾（Eugene Wong），男，1934年12月24日生于中国南京，计算机科学家、工程科学家，美国国家工程院院士、美国艺术与科学院院士、中国台湾中央研究院院士，加利福尼亚大学伯克利分校电气工程与计算机科学系荣休教授。\[1]\[2]\[7]\[9]

王佑曾于1955年、1958年、1959年在普林斯顿大学分别获得电气工程学士、硕士、博士学位；1959年—1960年任剑桥大学美国国家科学基金会博士后研究员；1960年—1962年任IBM约克敦研究中心研究员；1962年加入加利福尼亚大学伯克利分校电气工程与计算机科学系；1980年共同创办Relational Technology, Inc.，该公司后更名为INGRES Corporation；1985年—1989年任加利福尼亚大学伯克利分校电气工程与计算机科学系主任；1990年—1993年任美国白宫科技政策办公室副主任；1994年从加利福尼亚大学伯克利分校退休并受聘为荣休教授；1994年—1996年任香港科技大学研究与发展副校长；1998年—2000年任美国国家科学基金会工程助理主任。\[2]\[3]

王佑曾的研究方向为数据库管理系统、优化算法、随机过程和神经网络；其成果推动了随机过程、关系数据库和工程技术政策等领域的发展；曾获美国计算机协会软件系统奖、国际电气与电子工程师协会创始人奖章等荣誉，并当选为美国国家工程院院士、美国艺术与科学院院士、中国台湾中央研究院院士。\[1]\[2]\[4]\[7]\[8]

# 人物经历

1934年12月24日，王佑曾出生于中国南京。\[9]

1955年，王佑曾获得普林斯顿大学电气工程学士学位。\[2]

1958年，获得普林斯顿大学电气工程硕士学位。\[2]

1959年，获得普林斯顿大学电气工程博士学位。\[2]

1959年—1960年，任剑桥大学美国国家科学基金会博士后研究员。\[2]

1960年—1962年，任IBM约克敦研究中心研究员。\[2]

1962年，加入加利福尼亚大学伯克利分校电气工程与计算机科学系任教。\[2]

1980年，与迈克尔·斯通布雷克（Michael Stonebraker）、劳伦斯·A.罗（Lawrence A. Rowe）共同创办Relational Technology, Inc.，该公司后更名为INGRES Corporation。\[2]

1985年—1989年，任加利福尼亚大学伯克利分校电气工程与计算机科学系主任。\[2]

1990年—1993年，任美国白宫科技政策办公室副主任，参与推动高性能计算与通信、先进制造技术等联邦倡议，并参与《1991年高性能计算法案》成案版本相关工作。\[2]\[3]

1992年，当选为中国台湾中央研究院第19届工程科学组院士。\[1]

1994年，从加利福尼亚大学伯克利分校电气工程与计算机科学系退休，受聘为该系荣休教授。\[2]

1994年—1996年，任香港科技大学研究与发展副校长，推动技术转移和商业化相关项目建设。\[2]\[8]

1998年—2000年，任美国国家科学基金会工程助理主任。\[2]\[8]

1998年—2005年，在Versata, Inc.任董事、首席科学家、首席执行官等职务。\[2]

# 主要成就

## 科研成就

- **科研综述**

  王佑曾长期从事数据库管理系统、优化算法、随机过程和神经网络等研究。他在随机过程理论和数据库技术方面开展研究，围绕二维随机场、随机积分、随机微分方程、关系数据库查询处理、分布式数据库和动态再物化等问题发表成果；其与布鲁斯·哈耶克（Bruce Hajek）的研究为图像和多维数据处理提供了统计基础，其参与设计的INGRES系统成为早期关系数据库系统的重要代表。\[2]\[6]\[8]

- **科研奖励**

  | 获奖时间 | 类别 | 成果 |
  | :--- | :--- | :--- |
  | 1988年 | 美国计算机协会软件系统奖（ACM Software System Award） | INGRES关系数据库系统\[2]\[4] |

## 技术成就

王佑曾是INGRES关系数据库系统的共同设计者之一，并共同创办Relational Technology, Inc.，该公司后更名为INGRES Corporation，成为数据库软件产品提供商；INGRES关系数据库系统于1988年获美国计算机协会软件系统奖。1994年—1996年在中国香港期间，王佑曾先后以SuperNet, Ltd.首席执行官和Asia Internet Holding Company创办人的身份参与建设亚洲互联网骨干网络。\[2]\[3]\[4]

## 教育成就

- **培养成果**

  在加利福尼亚大学伯克利分校任教期间，王佑曾指导二十余篇博士论文。\[2]
