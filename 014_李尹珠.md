# 李尹珠

义项名：韩国首尔大学传播学教授、国际传播学会前会长

李尹珠（Eun-Ju Lee），女，韩国传播学者，国际传播学会会士（ICA Fellow），韩国首尔大学传播学教授、可信赖人工智能研究中心主任，曾任国际传播学会会长。\[1]\[2]\[4]\[5]

李尹珠具有斯坦福大学博士学术背景；曾任美国加州大学戴维斯分校教授；2009年—2012年任国际传播学会东亚区域主任；2017年—2020年任*Human Communication Research*主编；2019年当选为国际传播学会会士；2021年当选为国际传播学会会长候任当选人；2023年5月在加拿大多伦多举行的国际传播学会第73届年会上就任国际传播学会首位韩国籍会长。\[4]\[5]\[6]

李尹珠的研究方向为计算机中介传播、人机互动、媒体心理学、人工智能传播和数字传播效果；其成果推动了计算机中介传播与人机传播理论研究的发展；其论文曾获麦奎尔奖（McQuail Award），并当选为国际传播学会会士。\[2]\[5]\[7]\[8]

# 人物经历

2009年—2012年，李尹珠任国际传播学会东亚区域主任。\[5]

2017年—2020年，任国际传播学会官方期刊*Human Communication Research*主编，成为首位担任该刊主编的非英语国家学者。\[5]\[6]

2019年，当选为国际传播学会会士。\[5]

2021年，当选为国际传播学会会长候任当选人，成为首位获得该职位的亚洲女性学者。\[5]

2023年5月25日—29日，在加拿大多伦多举行的国际传播学会第73届年会上就任国际传播学会会长，成为首位韩国籍会长；其后，任韩国首尔大学传播学教授、可信赖人工智能研究中心主任。\[4]\[5]\[7]

# 主要成就

## 科研成就

- **科研综述**

  李尹珠长期围绕计算机中介传播、媒体心理学、人机互动和人工智能传播开展研究。其研究关注技术中介如何影响人们形成印象、处理信息并回应群体动态，也关注互联网新闻评论如何影响个体对新闻的感知和反应，以及社交媒体如何成为政治人物开展类社会互动的渠道。\[2]\[5]\[8]

  在研究轨迹上，李尹珠从一对一互动和小群体情境中的计算机中介传播研究，扩展至互联网新闻评论、政治人物社交媒体、错假信息、人机传播和人工智能传播研究。她提出面向大众导向计算机中介传播的真实性模型（Authenticity Model of CMC），并进一步提出人机传播整合模型（Integrative Model of Human-Machine Communication），用于解释技术中介内容的真实性感知、来源觉知和机器启发式（machine heuristic）等问题。\[2]\[5]\[7]
