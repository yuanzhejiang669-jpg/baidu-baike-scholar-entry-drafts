# 陈镇东

义项名：中国台湾海洋化学家、中国台湾中央研究院院士

陈镇东（Chen-Tung Arthur Chen，1949年—），男，出生于中国台湾省彰化县，海洋化学家，中国台湾中央研究院第34届数理科学组院士、美国地球物理联盟会士，任中国台湾中山大学海洋科学系约聘研究讲座教授。\[1]\[2]\[3]\[4]\[6]

陈镇东于1970年从中国台湾大学化学工程系毕业；1974年获得美国迈阿密大学海洋学院硕士学位；1977年获得该校化学海洋学博士学位；1977年任美国俄勒冈州立大学海洋学院助理教授；1981年任副教授；1984年任中国台湾中山大学海洋生物研究所客座教授；1985年任海洋地质研究所教授兼所长；1989年起任海洋科学相关院系教授，并先后兼任海洋科学学院院长、海洋科学研究中心主任；2006年任亚太海洋研究中心主任；2019年任海洋科学系约聘研究讲座教授；2022年当选美国地球物理联盟会士和国际整合海洋生物圈研究计划会士；2024年当选中国台湾中央研究院院士。\[2]\[3]\[4]

陈镇东主要研究海洋营养盐与碳循环、海洋酸化和全球变迁。他较早通过海洋碳测量研究人为二氧化碳在海洋中的吸收与储存，并对东海陆架营养盐来源、沿海海域碳交换、深海酸化及日本海深层水变化开展研究。\[5]\[3]\[7]

# 人物经历

1949年，陈镇东出生于中国台湾省彰化县。\[6]

1966年—1970年，在中国台湾大学化学工程系学习，获得学士学位。\[2]

1971年—1972年，在美国孟菲斯州立大学修读化学研究生课程。\[3]

1972年7月—1974年12月，在美国迈阿密大学海洋学院学习，获得硕士学位。\[2]\[3]

1975年1月—1977年6月，在美国迈阿密大学海洋学院学习，获得化学海洋学博士学位。\[2]\[3]

1977年7月，任美国俄勒冈州立大学海洋学院助理教授。\[2]

1981年7月，任美国俄勒冈州立大学海洋学院副教授。\[2]

1984年8月，任中国台湾中山大学海洋生物研究所客座教授。\[2]

1985年8月，任中国台湾中山大学海洋地质研究所教授兼所长。\[2]

1989年8月，任中国台湾中山大学海洋科学相关院系教授；同年起兼任海洋科学学院院长至1992年7月，并兼任海洋科学研究中心主任至1995年7月。\[2]

2006年8月—2011年7月，任中国台湾中山大学亚太海洋研究中心主任。\[2]

2008年—2015年，任国际科学理事会国际地圈生物圈计划副主席。\[2]\[3]

2019年7月，任中国台湾中山大学海洋科学系约聘研究讲座教授。\[2]\[3]

2022年，当选美国地球物理联盟会士；同年当选国际整合海洋生物圈研究计划会士。\[2]\[4]

2024年，当选中国台湾中央研究院第34届数理科学组院士。\[2]

# 主要成就

## 科研成就

- **科研综述**

陈镇东的研究集中于海洋营养盐和碳循环、海洋酸化、全球变迁以及边缘海生物地球化学。他在20世纪70年代研究深海碳酸钙与有机碳分解、海洋二氧化碳变化，较早利用海洋化学测量讨论人为二氧化碳在海洋中的吸收与储存问题。其后，他把研究范围拓展至大洋、边缘海和沿海生态系统，分析海气二氧化碳交换、陆架碳收支及营养盐输送过程。\[3]\[5]

陈镇东围绕西北太平洋及东亚边缘海开展研究，指出黑潮中层水是东海大陆架营养盐的重要来源，并研究渤海、黄海、东海的水团锋面与物质输送。他与合作者对全球沿海海域海气二氧化碳交换进行综合估算，讨论大陆架与近岸生态系统在碳循环中的不同作用。\[3]\[7]\[8]

在海洋酸化研究方面，陈镇东研究全球变暖、深层水缺氧和人为二氧化碳输入的共同影响。他与合作者发现，深海可能因变暖所引起的通气减弱和有机质分解而出现快于预期的酸化，并以日本海深层水为例分析其长期化学变化。\[3]\[9]
