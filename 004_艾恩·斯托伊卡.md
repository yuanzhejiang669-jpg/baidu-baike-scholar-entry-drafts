# 艾恩·斯托伊卡

义项名：加州大学伯克利分校电气工程与计算机科学系教授、Databricks联合创始人

艾恩·斯托伊卡（Ion Stoica），男，计算机科学家，美国国家工程院院士、国际计算机学会会士（ACM Fellow）、罗马尼亚科学院荣誉院士，加州大学伯克利分校电气工程与计算机科学系教授、徐葆校长讲席教授、Sky Computing Lab主任，Databricks联合创始人兼执行董事长。\[1\]\[2\]\[5\]\[11\]

艾恩·斯托伊卡于1989年毕业于布加勒斯特理工大学，获得计算机科学与控制工程硕士学位；2000年毕业于卡内基梅隆大学，获得电气与计算机工程博士学位；2006年共同创立Conviva；2013年共同创立Databricks；2019年共同创立Anyscale；2021年受聘为清华大学杰出访问教授；2024年当选为美国国家工程院院士。\[1\]\[2\]\[5\]\[11\]\[12\]

艾恩·斯托伊卡的研究方向为云计算、人工智能系统、网络与分布式计算机系统、服务质量和资源管理；其成果推动了分布式系统、云计算和大数据系统软件的发展。\[2\]\[5\]\[6\]\[7\]\[13\]

# 人物经历

1989年，艾恩·斯托伊卡毕业于布加勒斯特理工大学，获得计算机科学与控制工程硕士学位。\[2\]\[11\]

2000年，毕业于卡内基梅隆大学，获得电气与计算机工程博士学位。\[2\]\[11\]

2006年，共同创立Conviva，用于商业化大规模视频分发技术。\[2\]\[11\]

2012年，当选为国际计算机学会会士（ACM Fellow）。\[1\]\[2\]

2013年，共同创立Databricks，围绕Apache Spark相关大数据处理技术开展商业化。\[2\]\[11\]

2018年，作为首席研究员承担美国国家科学基金会Expeditions in Computing项目Secure, Real-Time Decisions on Live Data。\[3\]\[4\]

2019年，共同创立Anyscale。\[2\]\[12\]

2021年12月7日，受聘为清华大学杰出访问教授。\[1\]

2022年，当选为罗马尼亚科学院荣誉院士。\[1\]\[2\]

2024年，当选为美国国家工程院院士。\[5\]

# 主要成就

## 科研成就

- **科研综述**

  艾恩·斯托伊卡的研究主线包括网络与分布式计算机系统、互联网服务质量、资源管理、云计算和人工智能系统。他早期围绕可扩展服务质量体系、动态分组状态、Chord分布式哈希表等网络机制开展研究；在加州大学伯克利分校期间，其研究和系统工作覆盖Ray、Apache Spark、Apache Mesos、Tachyon、Chord DHT和Dynamic Packet State等系统。美国国家工程院将其当选理由概括为“面向大规模数据处理、分析和机器学习的网络化系统”，IEEE小林浩二计算机与通信奖授予理由为其对云和计算机网络服务设计的贡献。\[2\]\[5\]\[7\]

## 技术成就

艾恩·斯托伊卡主持或参与研发的Apache Spark、Apache Mesos、Tachyon、Chord DHT和Dynamic Packet State已成为大数据时代的基础性软件；其在AMPLab领导的软件系统工作包括Apache Spark、Apache Mesos和Tachyon，并通过共同创立Databricks推动Apache Spark相关大数据处理技术商业化。\[1\]\[11\]

艾恩·斯托伊卡共同创立的Anyscale围绕Ray开源项目构建计算平台，用于支持人工智能和机器学习应用扩展；其共同创立的Conviva则面向大规模视频分发技术商业化。\[11\]\[12\]
