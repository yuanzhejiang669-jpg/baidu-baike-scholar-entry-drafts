# 斯杰里马赫·亚历山大

斯杰里马赫·亚历山大（STELMAKH OLEKSANDR / Oleksandr Stelmakh），男，1965年9月生，乌克兰籍机械工程专家，中国工程院外籍院士，北京理工大学特聘教授、博士生导师。[1][2][5][12]

斯杰里马赫·亚历山大于1991年获得乌克兰国立航空大学博士学位；2019年入选我国海外高层次人才引进计划，全职在北京理工大学开展工作；2025年11月，当选中国工程院外籍院士。[1][2][5]

斯杰里马赫·亚历山大的研究方向为航空发动机摩擦润滑技术、机械摩擦与磨损、纳米摩擦学等；其研究涉及黏着-变形-流体动压摩擦磨损模型、润滑层挤压与稀疏、摩擦表面形貌等方向，推动了动力装备摩擦学和应用摩擦学研究进展；曾获中国政府友谊奖、北京市国际合作中关村奖等荣誉，并当选为中国工程院外籍院士。[1][2][5][6][8]

# 人物经历

1965年9月，斯杰里马赫·亚历山大出生。[12]

1991年，斯杰里马赫·亚历山大获得乌克兰国立航空大学博士学位，研究方向主要涉及机械摩擦与磨损、纳米摩擦学。[5]

2019年，入选我国海外高层次人才引进计划，全职在北京理工大学开展工作，任北京理工大学特聘教授、博士生导师，并面向先进动力摩擦学及纳米摩擦技术方向开展科研和人才培养工作。[1][3]

2025年11月21日，中国工程院公布2025年院士增选当选外籍院士名单；斯杰里马赫·亚历山大当选中国工程院外籍院士，国籍为乌克兰，研究方向为机械工程。[2]

# 主要成就

## 科研成就

- **科研综述**

  斯杰里马赫·亚历山大长期从事航空发动机摩擦润滑、动力装备摩擦学、纳米摩擦技术等研究，是航空发动机摩擦润滑技术专家，研究主要涉及机械摩擦与磨损、纳米摩擦学。[1][5]

  斯杰里马赫·亚历山大参与提出的黏着-变形-流体动压摩擦磨损模型，围绕曲面接触表面弹性变形过程、润滑层中挤压与稀疏过程、摩擦表面初始黏着和黏着磨损之间的关系展开，模型覆盖摩擦表面的载荷-速度范围和润滑模式。[6]

  在氢内燃机润滑研究方面，斯杰里马赫·亚历山大与合作者围绕含氢润滑环境下钢材料摩擦界面氧化物形态转换和磨损机制开展研究，相关成果发表于*Friction*。[5]

  在摩擦表面形貌研究方面，斯杰里马赫·亚历山大参与研究三维粗糙度对摩擦表面摩擦学效果的评价准则；相关论文讨论了黏着-变形-流体动压模型在摩擦磨损分析中的应用。[7]

  截至2025年11月，斯杰里马赫·亚历山大发表学术论文173篇，主编专著2部。[3]

- **发明专利**

  截至2025年11月，斯杰里马赫·亚历山大获授权专利65项。[3]

## 技术成就

斯杰里马赫·亚历山大围绕航空发动机摩擦润滑和纳米摩擦技术开展工程应用研究。其研发的磁脉冲湍流清洗装备已在航空企业得到应用；其行业会议报告还围绕黏着-变形-流体动压摩擦磨损理论及实践、提高内燃机效率的纳米摩擦学技术等应用主题展开。[8][10][11]

## 教育成就

- **人才培养**

  2019年起，斯杰里马赫·亚历山大全职在北京理工大学开展工作，投身中国动力装备摩擦学研究和人才培养事业；其所在团队面向先进动力摩擦学及纳米摩擦技术方向培养学生。[1][3]

# 社会生活

- **行业会议**

2019年，中国内燃机学会音视频资料收录斯杰里马赫·亚历山大在中国内燃机学会2019年联合学术年会的报告《提高内燃机效率的纳米摩擦学技术（俄文）》。[10]
