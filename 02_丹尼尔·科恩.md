# 丹尼尔·科恩

义项名：特拉维夫大学计算机科学与人工智能学院教授、ACM会士

丹尼尔·科恩（Daniel Cohen-Or），男，以色列计算机图形学家，计算机协会会士（ACM Fellow）、欧洲图形学协会会士（Eurographics Fellow），特拉维夫大学计算机科学与人工智能学院教授、视觉计算伊萨亚斯·尼兹里讲席教授。\[1\]\[4\]\[8\]\[9\]

丹尼尔·科恩于1985年以优异成绩获得本-古里安大学数学与计算机科学双学士学位；1986年以优异成绩获得该校计算机科学硕士学位；1991年获得纽约州立大学石溪分校计算机科学博士学位；其后在特拉维夫大学任教；2016年受聘为西蒙弗雷泽大学兼职教授；2021年当选为计算机协会会士。\[1\]\[4\]\[6\]\[9\]

丹尼尔·科恩的研究方向为计算机图形学、视觉计算、几何建模和生成模型；其成果推动了几何处理、形状分析、图像处理和图形学深度学习等领域的研究进展；曾获ACM SIGGRAPH计算机图形学成就奖、欧洲图形学杰出职业奖、欧洲图形学奖章等荣誉，并当选为计算机协会会士。\[1\]\[7\]\[8\]\[9\]

# 人物经历

1985年，丹尼尔·科恩以优异成绩获得本-古里安大学数学与计算机科学双学士学位。\[1\]\[4\]

1986年，以优异成绩获得本-古里安大学计算机科学硕士学位。\[1\]\[4\]

1991年，获得纽约州立大学石溪分校计算机科学博士学位；其后在特拉维夫大学计算机科学与人工智能学院任教。\[1\]\[4\]\[5\]

2016年，受聘为西蒙弗雷泽大学兼职教授。\[4\]\[6\]

2021年，当选为计算机协会会士（ACM Fellow）。\[8\]\[9\]

2024年，获欧洲图形学协会授予欧洲图形学奖章。\[8\]

# 主要成就

## 科研成就

- **科研综述**

  丹尼尔·科恩长期从事计算机图形学、视觉计算、几何建模和生成模型研究。他围绕点云、网格、体数据等数据表示形式开展算法研究，在几何处理、图形学与计算机视觉多个子方向形成了代表性成果。其“Laplacian Surface Editing”围绕局部微分性质保持开展曲面变形研究，“As-rigid-as-possible shape interpolation”将形状分解为单纯复形并通过仿射变换实现形状变形，“iWIRES: An Analyze-and-Edit Approach to Shape Manipulation”围绕人造对象中曲线结构的代数关系恢复全局结构。进入深度学习阶段后，他又围绕图形学应用发展了MeshCNN、点云几何变换网络和用于纹理、动画序列、三维形状合成的生成网络。相关成果影响了图像处理和几何处理等研究方向。\[7\]\[8\]

# 人物评价

丹尼尔·科恩在计算机图形学、几何处理和视觉计算方面作出了贡献*（“For contributions to computer graphics, geometry processing and visual computing.”）*。*（计算机协会评）*\[9\]
